1. Objetivos

Objetivo General

Familiarizarse con el uso de Batch Scripting en Windows, comprendiendo sus fundamentos básicos y reconociendo los beneficios que ofrece para la automatización de tareas, la optimización de procesos y la mejora de la eficiencia en entornos de trabajo.

Objetivos Específicos

  • Entender el uso de rutas relativas y variables dentro de la cmd.
  • Comprender las instrucciones básicas del Batch Scripting en Windows y su sintaxis general.
  • Analizar el uso de la instrucción @echo off y su importancia en la ejecución limpia de scripts.>
  • Comprender el uso de variables dentro de un script y su ejecucion en un script.
  • Identificar y aplicar mecanismos de input dentro de un script.
  • Conocer y utilizar estructuras de control básicas (condicionales y bucles) dentro de los script.

2. Investigación y Marco Conceptual

Bat Scripting en Windows es la forma más simple de automatizar tareas repetitivas. Con unos cuantos comandos puedes ejecutar procesos, gestionar archivos y ahorrar tiempo directamente desde la línea de comandos. Pequeño en código, grande en productividad. Existe también BASH scripting, el cual se ejecuta en terminales basades en UNIX (Linux y MAC)

Un archivo .BAT es un archivo de programación básica donde se va ejecutando linea por linea el script. No existen a diferencia de programas como C++ o Python, el script no tiene un interprete que revise si la syntaxis es correcta o si existe algunas fallas de lógica. 5. Conclusiones y Trabajo Futuro

En esta practica se exploraron comandos como md, tree, echo y el operador direcciónal. Mediante estos comando se familiarizo con como Windows interactua con el sistema.

Con esto en mente se pudo llegar a la conclusión que el sistema operativo funciona como un link entre el usuario y los componentes fisicos de la computadora. A través de comandos como dir, cd y md, el sistema operativo ofrece un metodo que facilitan la manipulación de la maquina sin que el usuario interactúe directamente con el hardware. El usuario no tiene que leer directamente la memoria del dispositivo y traducir 1 y 0 para descubrir que el archivo "hola.txt" contiene un hola.

Trabajo Futuro: Para este proyecto me gustaría explorar más comandos que vengan incluidos en la terminal de Windows. La documentación ofical ofrece información muy interesante e útil acerca de todas las funciones que viene incluidas.
