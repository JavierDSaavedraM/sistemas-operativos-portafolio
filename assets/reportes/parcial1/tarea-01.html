3. Desarrollo Técnico

Paso 1. Preparación del entorno.

Dentro de la terminal se utilizaron comandos para conocer la información del sistema: Version de Windows, Nombre del Usuario y Directorio del Usuarios. Para eso se ejectuaron los comandos de ver, echo %USERNAME% y echo %USERPROFILE%. El output se puede apreciar en la Figura 1.

Los comandos de echo utiliza las variables de entorno del sistema para poder desplegar esa información.

Paso 2. Creación de estructura de directorios.

Mediante el comando de md, se creo la carpeta SO_Lab1; Ubicada dentro del Escritorio. Usando igualmente md se pueden crear varias carpetas y subcarpetas en una sola linea. Para eso las subcarpetas deben ser escritas de manera continua con el \ separando.

Como se puede observar en la Figura 2, las carpetas dentro de Hardware son escritas al repetir Hardware\Procesadores Hardware\Memoria y Hardware\Expansion.

Finalmente se utiliza el comando tree /f. Esto muestra una vista del directorio en forma de arbol y al tener la bandera /f si existieran algunos archivos dentro igualmente se mostrarian.

Paso 3. Navegación y exploración

La Figura 3 muestra como dentro de la carpeta de .\Procesadores\, se busco retroceder dos nivele y acceder a la carpeta de Reportes. Al ejectura cd ..\..\ le dices al sistema que retroceda dos nivels y entonces te encontrarias en SO_LAB1 entonces puedes acceder a reportes.
