2. Investigación y Marco Conceptual

El sistem operativo ofrece diferentes abstracciónes para que el usuario pueda interactuar con el sistema. Esto facilita que el usuario no tenga que interactuar directametne con el hardware. Dentro de estas funcionalidades se encuentra el tasklist. Este permite que el usuario pueda ver los procesos que estan corriendo en el sistema de manera entendible para el ojo humano.

El sistema hace una lista de todos los procesos y muestra información como su nombre, identificar, memoria y recursos utilizados. Con la información presentada se pueden identificar los procesos que más estan consumiendo recursos y poder eliminarlos. De igual manera se puede identificar los procesos que tengan intenciónes maliciosas.

3. Desarrollo Técnico

Durante la clase se hablo de diferentes temas relaciondos a los sistemas operativos. Por esa razón se mostraron los comandos básicos para identificar la información del hardware instalado

Paso 1. Obtener información del sistema

Dentro de consola de comandos se puede utilizar systeminfo el cual obtiene la información directa de las especificaciónes de la maquina. Este ofrece información como: sistema opreativo; cantidad de espacio disponible y memoria RAM; arquitectura del CPU e información de los drivers. En la Figura 1 se muestra la información de mi sistema. Dentro de esto algo que el profe recalco mucho es la arquitectura del CPU. Aunque dentro de las especificaiones diga que es un Inel, la architectura real es de un AMD. Principalmente por ser computadoras portatiles de gamma media.

Paso 2. Obtener la información de los proceso corriendo en el sistema.

Con el comando tasklist se muestran todas las aplicaciones corriendo en el sistema. El comando puede tener diferentes parametros para obtener la información. Por ejemplo se pued combinar con /fi para filtrar los resultados a ciertos nombres, identifcadores, o cantidad de memoria. E incluso se puede hacer un sort de los resultados.

djjav@Windows-11:~/

tasklist /FI "IMAGENAME eq chrome.exe"

tasklist /FO CSV

tasklist /FI "MEMUSAGE gt 100000"

Paso 3. Combinación de comandos.

Todos estos comandos se pueden combinar de diferentes manera para obtener los resultados. Se puede combinar con more, less y find. Para facilitar la lectura del usuario.

El comando more hace que puedas ver una página a la vez. Mientras que el comando less te permite manejar el archivo. Usar las teclas y poder navegar por los resutlados de manera más sencilla.

Finalmente el comando find te ayuda a filtrar los resultados de cualquier comando. Al darle un string puede filtrar todo el text. La principal diferencia de este y el parametro /fi ya incluido en tasklist es que este te permite ignorar la capitalización de la cadena.

También cabe recalcar que se puede combinar con el operador redireccionador para poder guardarlo en un archivo.

djjav@Windows-11:~/

tasklist | find "chrome"

tasklist /FO CSV sort > Desktop\procesos.csv

tasklist /F0 list | more
