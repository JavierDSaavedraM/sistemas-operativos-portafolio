Mapeo de unidades virtuales con SUBST

Fecha: 15 de Enero, 2026

Autor: Javier Djorkaef Saavedra Mendiola

Tema: Windows y PATH

1. Objetivos

Objetivo General

Comprender la diferencia entre las unidades fisicas y las virtuales. Además de optimizar el flujo de trabajo con la creacion de unidades logicas virtuales.

Objetivos Específicos

  • Comprender la sintaxis del comando subst
  • Asociar la letra Z: a la ruta de la carpeta de sisop
  • Eliminar unidades virtuales de forma segura.

2. Investigación y Marco Conceptual

Para este ejercicio se utiliza el comando de subst. En los sistemas operativos Windows y DOS, el comando SUBST (Substitute) permite asociar una ruta de acceso (path) a una letra de unidad de disco.

"Una unidad virtual creada con SUBST se comporta ante el sistema y las aplicaciones como si fuera un disco físico o una partición real conectada al equipo."

3. Desarrollo Técnico

5. Conclusiones y Trabajo Futuro

En esta practica se logro explorar como crear unidades virtuales en Windows. Esto beneficia principalmente al desarrollo al poder tener un "atajo" de un directorio y no tener que estar escribiendo todo el path.

Una desventaja de esta funcion es que es por sesion. Si se sale de la terminal o se reinicia el sistema se pierde el atajo. Esto se puede evitar al configurar el ambiente para que al iniciar el equipo automaticamente se asigne.
